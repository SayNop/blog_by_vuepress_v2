# Blog by vuepress2
Blog based on vuepress v2

## visit online
click [here](https://saynop.github.io/)

## manual
put markdown files in *./detail* directory

## run & build
```shell
# before run
yarn install
# run
yarn blog:dev
# build
yarn blog:build
```

## node version
v16.20.0

## package
vuepress = "2.0.0"<br>

vuepress-plugin-blog2<br>

vuepress-plugin-copy-code2<br>

vuepress-plugin-comment2<br>

## update online
use update shell script
```bash
./update.sh
```
